# Employee-Attrition-Performance-Intelligence-System
project
## Project Objective

The objective of this project is to analyze employee attrition and performance
to identify key factors influencing employee turnover.

This project aims to:
- Analyze attrition trends using SQL and Excel
- Build interactive dashboards using Power BI
- Apply basic machine learning models to predict employee attrition
- Provide data-driven insights that can help HR teams improve retention strategies

## Business Questions

1. What is the overall employee attrition rate?
2. Which departments have the highest attrition?
3. Does salary and job level affect attrition?
4. Is attrition higher among employees with low performance ratings?
5. How does experience and tenure impact attrition?
6. Can we predict employee attrition using historical data?
