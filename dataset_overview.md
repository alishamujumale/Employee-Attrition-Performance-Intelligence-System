# Dataset Overview

## Dataset Name
Employee Attrition Dataset

## Source
Public HR Analytics dataset inspired by IBM Employee Attrition data

## Description
This dataset contains employee demographic, job-related, and performance
information used to analyze employee attrition patterns and predict attrition risk.

## Files
- raw/employee_attrition_raw.csv  
  → Original unprocessed dataset  

- processed/employee_attrition_cleaning.xlsx.csv  
  → Cleaned dataset after preprocessing

## Key Columns
- Age
- Attrition
- Department
- JobRole
- MonthlyIncome
- JobSatisfaction
- WorkLifeBalance
- YearsAtCompany

## Use Cases
- Attrition analysis
- HR performance insights
- Predictive modeling (Machine Learning)
- Dashboarding (Power BI / Tableau)
